# Homebrew Tap for renamed.to CLI

Official Homebrew tap for the [renamed.to](https://renamed.to) CLI tool.

## Installation

```bash
brew tap renamed-to/cli
brew install renamed
```

Or install directly:

```bash
brew install renamed-to/cli/renamed
```

## Usage

```bash
# Rename files using AI
renamed rename document.pdf

# Extract data from PDFs
renamed extract invoice.pdf

# Split PDFs
renamed pdf-split large-document.pdf

# Watch a directory for auto-organization
renamed watch ~/Downloads
```

## Updating

```bash
brew upgrade renamed
```
